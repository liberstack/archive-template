# História da Web

A World Wide Web foi criada em 1989 por Tim Berners-Lee, pesquisador do CERN (Organização Europeia para a Pesquisa Nuclear), na Suíça. O objetivo original era permitir que cientistas compartilhassem documentos e resultados de pesquisa de forma simples, através de uma rede de computadores já existente: a internet.

## O primeiro navegador e servidor

Em 1990, Berners-Lee desenvolveu o primeiro navegador web, chamado WorldWideWeb (posteriormente renomeado para Nexus), além do primeiro servidor web e da primeira página. Nesse mesmo período, ele definiu três tecnologias que ainda hoje são a base da web:

- **HTML** (HyperText Markup Language): linguagem de marcação para estruturar documentos.
- **URL** (Uniform Resource Locator): padrão de endereçamento para localizar recursos na rede.
- **HTTP** (HyperText Transfer Protocol): protocolo de comunicação entre cliente e servidor.

## Popularização nos anos 1990

A web se tornou pública em 1991 e cresceu lentamente até 1993, quando o navegador **Mosaic**, desenvolvido no NCSA (National Center for Supercomputing Applications), introduziu suporte a imagens dentro do texto. Isso tornou a navegação mais acessível a um público não técnico.

Em 1994, Berners-Lee fundou o **W3C** (World Wide Web Consortium), organização responsável por padronizar as tecnologias da web até os dias atuais. No mesmo ano, a Netscape Communications lançou o navegador Netscape Navigator, que dominou o mercado durante boa parte da década.

## HTML, CSS e a separação de responsabilidades

Nos anos seguintes, a web evoluiu de documentos estáticos para páginas com estilo visual separado do conteúdo. O **CSS** (Cascading Style Sheets), proposto em 1996, permitiu que a formatação visual fosse definida fora do HTML, facilitando a manutenção de sites maiores.

## A era dinâmica

No fim dos anos 1990 e início dos anos 2000, tecnologias como JavaScript, PHP e bancos de dados relacionais possibilitaram páginas dinâmicas, geradas a partir de dados armazenados em servidores. Esse período marcou a transição da web como um conjunto de documentos estáticos para uma plataforma de aplicações completas.

## Web 2.0 e a interatividade

A partir de meados dos anos 2000, o termo **Web 2.0** passou a descrever sites baseados em conteúdo gerado por usuários, redes sociais e aplicações que atualizavam partes da página sem recarregá-la por completo, usando a técnica conhecida como **AJAX** (Asynchronous JavaScript and XML).

## Padronização e HTML5

Em 2014, o W3C publicou oficialmente a especificação do **HTML5**, consolidando recursos como vídeo, áudio e canvas nativamente no navegador, sem depender de plugins externos como Flash.

Hoje a web continua evoluindo, com padrões abertos mantidos pelo W3C e pelo WHATWG (Web Hypertext Application Technology Working Group), garantindo que documentos publicados há décadas continuem acessíveis nos navegadores atuais.
